# Customer-Churn-Prediction-System-Using-Behavioral-Data-Analytics
This project is about predicting whether the customer is likely to churn using Machine Learning techniques. The system considers customer behavior data as input, predicts churn outcome using Random Forest classifier.

**Technologies Used**
Python
Pandas
NumPy
Scikit-learn
Streamlit
Matplotlib
Seaborn

**Features**
Customer churn prediction
Real-time dashboard interface
Probability score generation
Business insights and risk analysis

**Machine Learning Model**
Logistic Regression
Random Forest (Final Selected Model)

**Deployment**
The application interface was developed using Streamlit for interactive churn prediction analysis.
